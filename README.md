# 율의 시선: 다시 바라보는 하루

중학교 독서캠프용 선택형 스토리 웹게임 프레임워크입니다.
플레이어가 이름을 입력해 시작하고, 선택에 따라 기본 스탯, 인물 관계, 플래그, 다음 이벤트와 엔딩이 달라집니다.

## 파일 구성

- `index.html`: 화면 구조
- `style.css`: 디자인과 반응형 레이아웃
- `gameData.js`: 게임 제목, 스탯, 인물, 이벤트, 선택지, 엔딩 데이터
- `script.js`: 게임 진행, 조건 검사, 저장, 엔딩 판정 로직
- `assets/`: 장면별 SVG 이미지

## 실행 방법

1. 새 폴더를 만듭니다.
2. `index.html`, `style.css`, `gameData.js`, `script.js` 파일을 만듭니다.
3. `assets` 폴더를 만들고 SVG 파일들을 넣습니다.
4. `index.html`을 브라우저에서 엽니다.
5. GitHub Pages나 Netlify에 올리면 웹앱처럼 공유할 수 있습니다.
6. 나중에 학생 설계안을 반영하려면 `gameData.js`의 이벤트와 엔딩 데이터를 수정하면 됩니다.

## 교사용 수정 안내

게임 내용은 `gameData.js`에 모여 있습니다.
학생 설계안이 완성되면 `events` 배열의 장면, 선택지, 효과, 다음 이벤트 ID를 바꾸고, `endings` 배열의 엔딩 조건과 설명을 바꾸면 됩니다.
로직을 바꾸지 않아도 새 이야기로 확장할 수 있도록 구성했습니다.

## 저장 기능

게임 진행은 브라우저의 `localStorage`에 자동 저장됩니다.
새로고침 뒤 시작 화면에서 `이어하기` 버튼으로 저장된 진행을 불러올 수 있습니다.
`처음부터 다시 시작`을 누르면 확인창이 뜨고, 승인하면 저장된 진행이 삭제됩니다.
